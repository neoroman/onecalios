## Requirements
- iOS 13.0 or later
- Swift 5.0 or later
- Xcode 12.x

## Installation

OZChattingUI is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'OneCaliOS'
```
